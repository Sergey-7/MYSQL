
Курсовая работа

Спроектированная модель БД online shop предназначена для хранения данных интернет-магазина.
База данных включает 10 таблиц: каталог товаров (catalog), категории товаров (categories), бренды (brands), покупатели(customers),
анкеты покупателей (profiles), товары (products), изменение цены на товары (price_change), заказы (orders), скидки (discounts)
проданные товары (purchase_items).
БД позволяет помимо хранения всех внесенных данных вести учет купленных товаров (выполненных заказов), 
менять в таблице price_change цены на отдельные товары,которые будут обновляться в таблице purchase_items,
назначать скидки на конкретный товар для конкретного покупателя.
Заполнение таблиц осуществлялось с учетом отношений через сайт http://filldb.
зация.
Прилагаются скрипты сложных запросов для создания:
1. таблицы с колонками ФИО покупателя, его id, id заказа, название купленного товара, 
общая стоимость номенклатурного товара с учетом количества проданных единиц.
2. аналогичной таблицы с общей стоимостью товаров для каждого заказа, оплаченного покупателями. 

Прилагаются два представления (виртуальных таблицы). В одном представлении запросом выводятся данные о покупателе
и общей стоимостью оплаченных им товаров, в другом - можно увидеть данные о названии товара, его id и стоимости за единицу. 
Также прилагается скрин файла в формате ipynb (для Jupiter Notebook) как пример чтения данных из данной бд 
путем сложного запроса и преобразования их в таблицу DataFrame, которую можно использовать для обработки и подготовки данных
для дальнейшего анализа и построения моделей Data Science - http://joxi.ru/KAxdnkysZNgMRA
